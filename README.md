# Домашнее задание к занятию «Базы данных, их типы» Погребенко Александр

Задание 1. СУБД
Кейс
Крупная строительная компания, которая также занимается проектированием и девелопментом, решила создать правильную архитектуру для работы с данными. Ниже представлены задачи, которые необходимо решить для каждой предметной области.

Какие типы СУБД, на ваш взгляд, лучше всего подойдут для решения этих задач и почему?

1.1. Бюджетирование проектов с дальнейшим формированием финансовых аналитических отчётов и прогнозирования рисков. СУБД должна гарантировать целостность и чёткую структуру данных.
Ответ:
Реляционные СУБД (RDBMS) Примеры: PostgreSQL, MySQL, Microsoft SQL Server, Oracle. 
Почему подходят:
- Структурированность данных. Реляционные СУБД хорошо подходят для хранения структурированных данных, таких как проектная документация, сметы, данные о материалах, оборудовании, сотрудниках и т. д.. 
- Поддержка SQL. Язык SQL широко используется в строительных системах (CAD, ERP, CDE и др.) для создания, модификации и управления данными. 
- Интеграция с другими системами. Реляционные СУБД легко интегрируются с веб-приложениями, ERP-системами (например, «1С»), CRM и другими инструментами, которые часто применяются в строительстве. 
- Зрелость и надёжность. Многие реляционные СУБД имеют долгую историю развития, обширную документацию и большое сообщество пользователей.
